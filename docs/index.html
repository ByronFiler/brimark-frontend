brimark-frontend

  1. Download nodeJS v14.15.3

  2. Download the parcel-bundler:

    npm install -g parcel-bundler@1.12.4

  3. Install Typescript:

    npm install -g typescript@4.1.5

  4. To install all the packages from the package.json:

    npm install

  5. To fix any severity vulnerabilities:

    npm audit fix

To run the project locally use the command. Note: this will run the tests first by default and then run the server!

npm run dev

To run the tests:

npm test

If you want to build the project:

npm run build

To build the documentation:

npm run ts-docs

To build the test coverage document:

npm run ts-coverage
